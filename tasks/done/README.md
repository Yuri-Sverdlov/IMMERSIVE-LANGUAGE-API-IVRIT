# tasks/done — архив принятых заданий

Сюда архитектор перекладывает принятые задания после проверки.

Соглашение: на каждое задание — подпапка `NNN-краткое-имя/` с тремя файлами:

- `TASK.md` — что просили (копия активного TASK на момент приёмки);
- `REPORT.md` — что сделал кодер;
- `ACCEPTED.md` — вердикт архитектора: что принято, замечания, дата.

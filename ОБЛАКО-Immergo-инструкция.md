# Immergo — запуск через облако (Google Cloud Run)

Это веб-приложение (фронтенд Vite + бэкенд FastAPI), которое работает как «пульт» к
**Google Gemini Live API**. Локально ничего ставить не нужно — всё собирается и крутится
в Google Cloud. Ниже — самый простой путь: развёртывание в **Cloud Run** в один клик.

---

## 0. Что нужно заранее
1. **Аккаунт Google** (обычный Gmail подойдёт).
2. **Проект Google Cloud с включённой оплатой (Billing)** — привязанная банковская карта.
   Без billing Gemini Live и Cloud Run не запустятся.
   Создать проект: https://console.cloud.google.com/projectcreate
   Включить оплату: https://console.cloud.google.com/billing
3. Понимание стоимости: Gemini Live **платный по факту использования**,
   ориентир ~**1,7 цента за минуту** разговора. См. раздел 5 (как не словить лишние счёта).

---

## 1. Развёртывание в один клик (рекомендуется)
1. Открой страницу репозитория: https://github.com/ZackAkil/immersive-language-learning-with-live-api
2. Нажми кнопку **«Run on Google Cloud»** (в начале README).
3. Откроется **Google Cloud Shell** в браузере. Согласись с авторизацией (Authorize).
4. Выбери свой **проект** (тот, где включён billing) и **регион** (например, `us-central1`).
5. Скрипт автоматически:
   - включит API: `aiplatform.googleapis.com`, `artifactregistry.googleapis.com`;
   - выдаст сервисному аккаунту роль `aiplatform.user`;
   - соберёт контейнер по `Dockerfile` (фронтенд + бэкенд);
   - развернёт сервис в **Cloud Run**.
6. По завершении в консоли появится **URL вида** `https://immersive-...-uc.a.run.app`.
   Открой его в браузере (Chrome) — это и есть твоё приложение.

> Если на каком-то шаге попросит «enable API / привязать billing» — соглашайся, это нормально.

---

## 2. Первый запуск в браузере
1. Открой выданный `*.run.app` URL.
2. Браузер попросит доступ к **микрофону** — разрешить.
3. В приложении:
   - выбери **родной язык** (Russian) — на нём будут подсказки/переводы;
   - выбери **сценарий-миссию** (заказ кофе, билет на автобус и т.п.);
   - выбери **режим**:
     - **Teacher Mode** — помогает, объясняет и переводит на родной язык;
     - **Immersive Mode** — строго: продвигаешься, только говоря на изучаемом языке;
   - начинай говорить — ИИ отвечает голосом в реальном времени и оценивает беглость.

### Про иврит
Изучаемый язык задаётся внутри приложения/сценария; родной ставь Russian.
Иврит — RTL; если в каком-то сценарии иврит не предложен, можно «вайб-кодингом»
добавить свой сценарий (см. раздел GEMINI.md в репозитории), но это уже доработка.

---

## 3. Где всё это «живёт» в Google Cloud
- **Cloud Run** — сам сервис (запущенное приложение): https://console.cloud.google.com/run
- **Vertex AI** — собственно Gemini Live (туда уходят запросы).
- **Artifact Registry** — хранит собранный образ контейнера.
URL по умолчанию **публичный** (`--allow-unauthenticated`) — кто знает ссылку, может открыть.
Для личного использования просто не делись ссылкой.

---

## 4. Обновление / повторное развёртывание
Кнопка ставит версию на момент клика. Чтобы обновиться позже — снова нажми кнопку,
либо разверни из терминала (нужен установленный `gcloud`):
```
cp scripts/example.deploy.sh scripts/deploy.sh   # отредактируй PROJECT_ID и т.д.
./scripts/deploy.sh
```

---

## 5. ВАЖНО про деньги (чтобы не было сюрпризов)
- Cloud Run **сам по себе** в простое стоит почти ноль (масштабируется в 0), платишь
  в основном за **минуты разговора с Gemini** (Vertex AI).
- В конфиге есть лимит сессии `SESSION_TIME_LIMIT` (по умолчанию 180 сек) и
  ограничения частоты — они защищают от перерасхода.
- **Поставь бюджет-алерт**: https://console.cloud.google.com/billing → Budgets & alerts
  (например, $5 в месяц с уведомлением на почту).
- Когда наиграешься и хочешь гарантированно ноль трат — **удали сервис Cloud Run**:
  Console → Cloud Run → выбрать сервис → Delete. Развернуть заново можно в любой момент.

---

## 6. Подводные камни
- **Нет billing** → развёртывание/запросы к Gemini падают. Сначала включи оплату.
- **Регион**: держи `LOCATION`/регион одинаковым (например, `us-central1`).
- **Микрофон**: нужен HTTPS (у `*.run.app` он есть) и разрешение в браузере.
- **Публичный URL**: не выкладывай ссылку, если не хочешь, чтобы расходовали твой лимит.
- **reCAPTCHA / Redis / BigQuery** — опциональные «продакшн»-фичи, для личного теста не нужны.

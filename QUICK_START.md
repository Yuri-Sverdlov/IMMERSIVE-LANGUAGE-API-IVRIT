# QUICK_START — быстрый старт для пользователя

> Для **первой установки** на новом компьютере см. техническую инструкцию **`DEPLOY.md`**.
> Этот файл — для **ежедневного использования**: что нажимать и какие есть возможности.

## Что это

**Immergo (наша версия)** — голосовой тренажёр языка. Вы выбираете сценарий (квест),
говорите в микрофон, а ИИ отвечает **голосом** в роли собеседника (продавец, сосед, барista…).

- Работает **локально** на вашем ПК (браузер `localhost`).
- «Мозг» (Gemini) — в **интернете** Google; без интернета разговор не пойдёт.
- Настроено по умолчанию: **родной язык — русский**, **изучаемый — иврит**.

---

## Первый раз на этом компьютере

1. Клонировать репозиторий и установить зависимости — по **`DEPLOY.md`** (Node, Python, `npm install`, venv).
2. Получить ключ **GEMINI_API_KEY** на https://aistudio.google.com (кнопка *Get API key*).
3. Скопировать `app/.env.example` → `app/.env` и вписать ключ:
   ```
   GEMINI_API_KEY=ваш-ключ-здесь
   ```
4. **Важно:** в файле `.env` пишите только **латиницу/ASCII** в комментариях (кириллица в `.env`
   может сломать запуск на Windows).

После этого каждый день достаточно **`start-immergo.bat`**.

---

## Ежедневный запуск (Windows)

Дважды щёлкните **`start-immergo.bat`** в корне проекта.

Скрипт:
1. Освобождает порты **8000** и **5173**, если они заняты старыми процессами.
2. Открывает окно **Immergo Backend** (сервер, логи, токены).
3. Открывает окно **Immergo Frontend** (интерфейс).
4. Через ~8 секунд открывает браузер: **http://localhost:5173/**

**Остановка:** закройте оба окна терминала (Backend и Frontend).

## Экран выбора миссии

### Языки (верхняя карточка)

| Поле | Значение по умолчанию |
|---|---|
| **I speak / Я говорю** | Russian (русский) |
| **I want to learn / Я хочу учить** | Hebrew (иврит) |

Можно сменить языки в выпадающих списках **до** выбора квеста.

> Если браузер переводит страницу на русский — подписи могут выглядеть по-русски,
> но внутри всё равно работает выбранный язык (иврит/русский).

### Режим (два варианта)

| Режим | Для кого | Поведение |
|---|---|---|
| **Teacher / Режим учителя** | Начинающие | Помогает, переводит на русский, мягко исправляет |
| **Immersive / Иммерсивная ролевая игра** | Продвинутые | Строго: нужно говорить на изучаемом языке, «без поблажек» |

Выберите режим **до** клика по квесту.

### Квесты (список карточек)

Каждая карточка — сценарий с уровнем **Easy / Medium / Hard / Expert**.
Бейдж сложности **косметический** (на поведение ИИ почти не влияет) — ориентируйтесь на описание.

Нажмите карточку — начнётся сессия.

## Во время миссии

1. Браузер попросит **доступ к микрофону** — разрешите (один раз на сайт).
2. Статус внизу: *Connected and ready* / *Связаны и готовы* — можно говорить.
3. Говорите на **иврите** (в Immersive) или как подсказывает учитель (в Teacher).
4. ИИ отвечает **голосом** — дождитесь паузы (~2 сек тишины после вашей реплики).
5. Красная кнопка **End mission / Завершить миссию** — закончить досрочно.

### Если ИИ молчит

- Проверьте окно **Immergo Backend** — нет ли ошибок (красный текст).
- Обновите страницу (`Ctrl+F5`) и начните миссию заново.
- Убедитесь, что в настройках выбран **Hebrew**, а не другой язык.

### Если ответы на «не том» языке

- Обновите страницу (`Ctrl+F5`) — сбросит устаревшие настройки в браузере.
- Перед миссией проверьте оба выпадающих списка языков.

## После миссии

Экран итогов:
- **Успех** — миссия выполнена, оценка (Tiro / Proficiens / Peritus) и советы на русском.
- **Не выполнена** — «Ты не выполнил задачи…» — закончили раньше времени или не достигли цели сценария.

Кнопка **Return to mission list** — вернуться к выбору квестов.

## Расход токенов (квота Google)

В окне **Immergo Backend** после ответов ИИ появляются строки:

```
[TOKENS] total=1420 prompt=1266 response=154
```

| Поле | Значение |
|---|---|
| `total` | Всего токенов за этот ответ |
| `prompt` | Вход (ваша речь + контекст) |
| `response` | Выход (ответ ИИ) |

**Сколько осталось до лимита** — приложение не показывает; смотрите квоту в Google AI Studio.
По `total` за сессию можно прикинуть расход.

## Ограничения (важно знать)

| Параметр | Сейчас | Где менять |
|---|---|---|
| Длительность одной сессии | **7 минут** (420 сек) | `app/.env` → `SESSION_TIME_LIMIT=420` |
| Пауза тишины перед ответом ИИ | **~2 секунды** | в коде фронта (скоро — в config-файле) |
| Языки по умолчанию | RU → HE | выпадающие списки + localStorage браузера |

После 7 минут сервер **сам закроет** сессию — это нормально, начните новую миссию.

## Частые проблемы

| Симптом | Что делать |
|---|---|
| `start-immergo.bat` пишет *venv not found* | Выполните установку из **`DEPLOY.md`** |
| Порт занят / страница не открывается | Запустите `start-immergo.bat` снова — он освобождает порты |
| Нет голоса от ИИ | Смотрите лог Backend; проверьте ключ в `.env` |
| ИИ говорит на французском | `Ctrl+F5`, проверьте Hebrew в списке «I want to learn» |
| Ошибка при старте backend | Проверьте `.env`: комментарии только на **английском (ASCII)** |

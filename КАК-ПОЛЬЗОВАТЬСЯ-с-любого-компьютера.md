# Immergo — как пользоваться с любого компьютера

## Главное
Приложение уже развёрнуто в облаке (Google Cloud Run). Оно НЕ на твоём компьютере.
Чтобы пользоваться им с любого устройства — **ничего скачивать и настраивать не нужно**.

## Чтобы пользоваться (на любом ПК / телефоне)
1. Открой в браузере (Chrome) адрес:
   **https://immergo-581555965916.us-central1.run.app**
2. Нажми **Start**, разреши доступ к **микрофону**.
3. Выбери родной язык, сценарий, режим — и говори.

Добавь адрес в закладки. Входить в Google-аккаунт для использования НЕ требуется
(приложение публичное). Никакого репозитория, Node, Python, gcloud — ничего этого
на новом компьютере не нужно.

> Не делись этой ссылкой: URL публичный, чужие сессии тратят твой лимит Google Cloud.

## Нужно ли скачивать репозиторий / создавать репозиторий на GitHub?
**Нет** — ни для использования, ни для повторного запуска.
- Репозиторий (исходники) нужен только если захочешь **изменить код** или **развернуть заново**.
- Свой репозиторий на GitHub нужен только если захочешь **сохранять свои правки** кода.
  Сейчас в этом нет необходимости — мы развернули из публичного репозитория автора.

## Если когда-нибудь понадобится развернуть заново / обновить
Это тоже делается в браузере, без установки на компьютер:
1. Зайди в Google Cloud → выбери проект **immergo-hebrew**.
2. Открой **Cloud Shell** (иконка терминала вверху справа).
3. Выполни (репозиторий клонируется в облаке):
   ```
   cloudshell_open --repo_url "https://github.com/ZackAkil/immersive-language-learning-with-live-api" --page editor --force_new_clone
   cd ~/cloudshell_open/immersive-language-learning-with-live-api
   gcloud run deploy immergo --source . --region us-central1 --allow-unauthenticated --quiet \
     --set-env-vars PROJECT_ID=immergo-hebrew,LOCATION=us-central1,MODEL=gemini-live-2.5-flash-native-audio,DEV_MODE=true
   ```

## Деньги / как остановить траты
- Платишь только за минуты разговора (~1,7 ¢/мин). Бюджет-алерт ₪20/мес уже стоит.
- Сервис Cloud Run в простое масштабируется в ноль (почти не стоит), но для
  гарантированного нуля трат — удали его.

### Как удалить сервис в Cloud Run (подробно, через браузер)
1. Открой https://console.cloud.google.com/run
2. Вверху страницы проверь, что выбран **проект `immergo-hebrew`**
   (если нет — кликни селектор проекта вверху и выбери его).
3. В списке сервисов найди строку **`immergo`**.
4. Поставь **галочку слева** от `immergo` (или открой сервис кликом по имени).
5. Вверху нажми кнопку **DELETE / УДАЛИТЬ** (иконка корзины).
6. Появится окно подтверждения — оно может попросить **ввести имя сервиса**
   (`immergo`) для подтверждения. Введи и нажми **DELETE / УДАЛИТЬ**.
7. Через несколько секунд сервис исчезнет из списка. Публичный URL перестанет работать —
   это и значит, что трат по сервису больше нет.

### Альтернатива — одной командой в Cloud Shell
```
gcloud run services delete immergo --region us-central1 --quiet
```

### Что важно понимать
- Удаляется **только сервис**, а не проект и не биллинг. Гемини тарифицируется
  по факту запросов, так что без работающего сервиса новых трат не возникает.
- Это **обратимо**: развернуть заново можно в любой момент по инструкции
  «Если понадобится развернуть заново» выше — данные приложения не теряются
  (оно без базы, состояние не хранит).
- Можно НЕ удалять: в простое сервис почти ничего не стоит. Удаление — это «на всякий случай».

## Параметры развёртывания (для справки)
- Проект: `immergo-hebrew`
- Сервис Cloud Run: `immergo`, регион `us-central1`
- Модель: `gemini-live-2.5-flash-native-audio`
- Режим: Simple/DEV_MODE=true (без reCAPTCHA и Redis — норм для личного использования)
- Репозиторий-источник: https://github.com/ZackAkil/immersive-language-learning-with-live-api

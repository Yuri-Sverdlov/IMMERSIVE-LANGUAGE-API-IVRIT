# Как добавить свой сценарий (квест)

> Вы можете добавить любой разговорный сценарий без написания кода — достаточно отредактировать один JSON-файл.

---

## Зачем

Приложение поставляется с готовыми квестами (кофе, автобус, собеседование…). Если хотите практиковать
что-то своё — напр. объяснить внуку правила шашек, поторговаться на рынке, заказать что-то по телефону —
просто добавьте свой объект в список миссий.

---

## Где лежит список квестов

Файл: **`app/src/data/missions.json`**

Это обычный JSON-массив (`[...]`) из объектов — по одному на каждый квест.

---

## Пошагово

### 1. Откройте шаблон

Файл: **`app/src/data/mission.template.json`**

Содержимое:
```json
{
  "id": 10,
  "title": "Your mission title",
  "difficulty": "Easy",
  "desc": "What the learner should achieve in this scenario.",
  "target_role": "Who the AI plays, e.g. Friendly Shopkeeper"
}
```

### 2. Скопируйте объект и заполните свои значения

Откройте шаблон любым текстовым редактором (Блокнот, VS Code и т.д.),
скопируйте весь блок `{ … }` и заполните поля (подробности — в таблице ниже).

### 3. Вставьте в `missions.json`

Откройте `app/src/data/missions.json`. Найдите конец массива — закрывающую `]`.
Перед ней добавьте запятую после предыдущего объекта и вставьте ваш блок:

```json
[
  { ... существующая миссия ... },
  { ... ещё одна ... },
  {
    "id": 10,
    "title": "Научить внука играть в шашки",
    "difficulty": "Easy",
    "desc": "Твоя задача — объяснить внуку правила шашек и сыграть первую партию.",
    "target_role": "Любознательный внук"
  }
]
```

> **Важно:** запятая ставится **после** каждого объекта, кроме последнего.

### 4. Сохраните файл

Сохраните `missions.json` (Ctrl+S или File → Save).

### 5. Обновите браузер

- Если dev-сервер уже запущен (`start-immergo.bat`) — нажмите **Ctrl+F5** в браузере.
- Если dev-сервер не подхватил — закройте оба окна терминала и запустите `start-immergo.bat` заново.

Ваш квест появится в списке миссий.

## Таблица полей

| Поле | Что значит | Пример |
|------|-----------|--------|
| `id` | Уникальный номер. **Не повторяйте** существующие (сейчас заняты: 0–6, 8, 9). | `10` |
| `title` | Название квеста на карточке и в промпте ИИ. | `"Научить внука играть в шашки"` |
| `difficulty` | Только **бейдж** на карточке — `Easy`, `Medium`, `Hard` или `Expert`. **На поведение ИИ не влияет.** | `"Easy"` |
| `desc` | Цель ученика — что нужно сделать/достичь в сценарии. Попадает в промпт ИИ. | `"Объясни правила шашек и сыграй первую партию."` |
| `target_role` | Кем ИИ будет в разговоре. Попадает в промпт («Roleplay: …»). | `"Любознательный внук"` |

> `desc` и `target_role` можно писать **на русском или на иврите** — ИИ поймёт оба языка.
> Язык разговора задаётся в UI и `immergo.config.json`, не здесь.

---

## Пример: «Научить внука играть в шашки»

Готовый блок для вставки:

```json
{
  "id": 10,
  "title": "Teach grandson to play checkers",
  "difficulty": "Easy",
  "desc": "Explain the rules of checkers and play a first game together.",
  "target_role": "Curious Grandson"
}
```

Или полностью на русском:

```json
{
  "id": 10,
  "title": "Научить внука играть в шашки",
  "difficulty": "Easy",
  "desc": "Объясни внуку правила шашек и сыграй с ним первую партию.",
  "target_role": "Любознательный внук"
}
```

## Частые ошибки

| Ошибка | Почему | Как исправить |
|--------|--------|---------------|
| Квест не появился в списке | Невалидный JSON | Проверьте файл на сайте [jsonlint.com](https://jsonlint.com) или запустите `npm run build` из папки `app/` |
| Дублирующий `id` | Два квеста с одинаковым номером | Дайте новому квесту уникальный `id` (напр. 10, 11, 12…) |
| Пропущенная запятая | Между объектами в массиве нужна `,` | Добавьте `,` после закрывающей `}` предыдущего объекта |
| Лишняя запятая | Нельзя ставить `,` после **последнего** объекта перед `]` | Удалите запятую |
| `difficulty` написан не так | Бейдж не окрасится | Используйте точно: `Easy`, `Medium`, `Hard`, `Expert` |

---

## Языки

Язык, на котором ИИ говорит (иврит) и ваш родной язык (русский) задаются:
- в выпадающих списках на экране выбора миссии;
- в файле **`app/immergo.config.json`** (поля `target_language`, `native_language`).

В самой миссии (`missions.json`) язык **не задаётся**.
